# MNIST

仅使用numpy手动构造的手写数字识别

所用BP神经网络算法参见（建议下载后用Typora打开）：[《BP神经网络》](https://github.com/FvTao/Notes/blob/master/Knowledge/BP%E7%A5%9E%E7%BB%8F%E7%BD%91%E7%BB%9C.md)

# Usage

1. 安装`Python>=3.10`
2. 执行以下命令
   ```bash
   git clone https://github.com/FvTao/MNIST.git
   cd MNIST
   pip3 install numpy
   python3 main.py
   ```
